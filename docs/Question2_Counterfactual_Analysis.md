# Question 2: Counterfactual Analysis - Method Comparison

## 1. 核心问题

每个赛季只有其中一种投票结合方法的淘汰结果，另一种方法的结果是缺失的：
- **第1-2季**：排名法 (Rank)
- **第3-27季**：百分比法 (Percent)
- **第28-34季**：排名法 (Rank) + 评委从垫底两人中选择

## 2. 解决方案

利用Question 1中MCMC估计的粉丝投票份额，对每一周重新计算两种方法的淘汰结果，进行**反事实模拟 (Counterfactual Simulation)**。

### 2.1 两种方法的计算规则

**排名法 (Rank Method)**:
1. 评委分数排名：分数越高 → 排名越前（数字越小）
2. 粉丝投票排名：票数越高 → 排名越前
3. 两排名相加，**总排名最高（数字最大）者淘汰**

**百分比法 (Percent Method)**:
1. 评委百分比 = 个人分数 / 所有人分数之和
2. 粉丝百分比 = 个人票数 / 所有人票数之和
3. 两百分比相加，**总百分比最低者淘汰**

## 3. 验证结果

### 3.1 MCMC一致性验证

| 赛季范围 | 实际使用方法 | 预测与实际一致率 |
|---------|-------------|-----------------|
| 1-2, 28-34 | 排名法 | 91.2% (52/57) |
| 3-27 | 百分比法 | **100%** (168/168) |

### 3.2 不一致周分析（5周）

所有不一致都发生在第28季之后，原因是**评委选择机制**：

| 赛季-周 | 预测淘汰 | 实际淘汰 | 实际在垫底两人中？ |
|--------|---------|---------|------------------|
| S28W6 | Sean Spicer | Sailor Brinkley-Cook | ✓ Yes |
| S29W6 | Skai Jackson | Veron Davis | ✓ Yes |
| S30W5 | Kenya Moore | Melanie C | ✓ Yes |
| S32W6 | Harry Jowsey | Mauricio Umansky | ✓ Yes |
| S34W4 | Dylan Efron | Hilaria Baldwin | ✓ Yes |

**结论**：所有实际淘汰者都在垫底两人中，证实了评委选择机制的存在。

## 4. 两种方法的差异

### 4.1 总体差异率

- 总分析周数：225周
- 两种方法导致不同淘汰：13周 (**5.8%**)

### 4.2 粉丝偏好分析

定义"粉丝最爱"：评委排名靠后但粉丝排名靠前的选手

| 指标 | 排名法 | 百分比法 |
|-----|-------|---------|
| 粉丝最爱存活率 | 99.5% | **100%** |

**结论**：百分比法略微更偏向粉丝投票（+0.5%）

## 5. 争议选手分析

### 5.1 Bristol Palin (第11季，第3名)

- 参与9周比赛
- **两种方法下都不会被淘汰** (0次)
- 粉丝排名始终第1（粉丝支持极强）

### 5.2 Bobby Bones (第27季，冠军)

- 参与5周比赛
- **两种方法下都不会被淘汰** (0次)
- 第4周出现差异：Rank→Evanna Lynch，Pct→Tinashe
- 粉丝排名始终前2

### 5.3 关键发现

争议选手的粉丝支持度极高，**无论使用哪种方法都不会改变他们的结局**。问题不在于方法选择，而在于粉丝投票与评委评分的根本性脱节。

## 6. 结论与建议

### 6.1 方法选择

1. **两种方法差异很小**（仅5.8%不同）
2. **百分比法更偏向粉丝**，但差异微乎其微
3. 对于极端争议案例（如Bobby Bones），**方法选择无法改变结果**

### 6.2 评委选择机制

第28季起引入的"评委从垫底两人中选择"机制：
- 为评委保留了最终决定权
- 可以纠正约8.8%的"粉丝主导"淘汰
- 实际使用率约为8.8% (5/57周)

### 6.3 建议

1. **保留排名法**：简单透明，与百分比法差异小
2. **保留评委选择机制**：提供安全阀，但谨慎使用
3. **根本解决需要新机制**：见Question 4的混合权重系统
