# Question 2: Counterfactual Analysis - Method Comparison

## 1. 核心问题

每个赛季只有其中一种投票结合方法的淘汰结果，另一种方法的结果是缺失的：
- **第1-2季**：排名法 (Rank)
- **第3-27季**：百分比法 (Percent)
- **第28-34季**：排名法 (Rank) + 评委从垫底两人中选择

## 2. 解决方案

利用Question 1中MCMC估计的粉丝投票份额，对每一周重新计算两种方法的淘汰结果，进行**反事实模拟 (Counterfactual Simulation)**。

### 2.1 两种方法的计算规则

**排名法 (Rank Method)**:
1. 评委分数排名：分数越高 → 排名越前（数字越小）
2. 粉丝投票排名：票数越高 → 排名越前
3. 两排名相加，**总排名最高（数字最大）者淘汰**

**百分比法 (Percent Method)**:
1. 评委百分比 = 个人分数 / 所有人分数之和
2. 粉丝百分比 = 个人票数 / 所有人票数之和
3. 两百分比相加，**总百分比最低者淘汰**

## 3. 分析结果概览

### 3.1 总体差异统计 (STEP 2-3)

| 指标 | 数值 |
|------|------|
| 分析周数 | 276周 |
| 两种方法结果一致 | 192周 (69.6%) |
| 两种方法结果不同 | **84周 (30.4%)** |

### 3.2 各赛季差异率 (STEP 3)

差异率最高的5个赛季：

| 赛季 | 实际使用方法 | 差异周数 | 差异率 |
|------|-------------|---------|--------|
| Season 33 | 排名法 | 5/7 | **71.4%** |
| Season 30 | 排名法 | 6/9 | 66.7% |
| Season 2 | 排名法 | 5/8 | 62.5% |
| Season 20 | 百分比法 | 5/8 | 62.5% |
| Season 3 | 百分比法 | 4/7 | 57.1% |

**观察**：S28后的排名法赛季差异率普遍较高。

## 7. 最终结论与建议 (STEP 8)

### 7.1 两种方法的差异

- 在**30.4%**的周中，两种方法产生不同淘汰结果
- 百分比法略微更偏向粉丝投票 (+1.2%)

### 7.2 争议选手分析

- Jerry Rice (S2): 粉丝排名#1-2，两种方法下均存活到决赛
- Billy Ray Cyrus (S4): 类似情况，粉丝支持使其获得第5名
- Bristol Palin (S11): 粉丝排名全部#1，两种方法下均不会被淘汰
- Bobby Bones (S27): 粉丝排名#2，依靠粉丝投票赢得冠军

**★ 核心发现**: 争议的根本原因是粉丝投票与评委分数的脱钩，而非方法选择

### 7.3 评委选择机制对争议选手的影响

- 如果对Jerry Rice等采用评委选择机制，评委可能更早淘汰他们
- 该机制赋予评委"最后决定权"，可推翻纯粉丝投票的结果
- 这正是S28后引入该机制的设计目的：防止争议情况再现

### 7.4 建议使用的方法

**★ 推荐使用【百分比法】**

理由：
1. **更透明**: 粉丝知道自己的票"值多少"
2. **更公平**: 避免了排名法中"一票之差=一名之差"的不合理放大效应
3. **更稳定**: 历史上(S3-S27)运行良好

### 7.5 是否建议评委选择机制?

**★ 建议【保留】评委从垫底两人中选择的机制**

理由：
1. 给予专业评委最终话语权，平衡粉丝投票的潜在非理性
2. 可有效防止"Bobby Bones式争议"再次发生
3. 增加节目的戏剧性和悬念
