# 第三题：因素影响分析 - 方法选择与实验报告

## 1. 研究目标

分析影响《与星共舞》(DWTS) 中**评委评分**和**粉丝投票**的因素差异。

**核心问题**：职业舞伴(Partner)、名人行业(Industry)、年龄(Age)、赛季(Season)等因素对评委评分和粉丝投票的影响方式是否相同？

## 2. 数据准备

### 2.1 数据来源

| 数据集 | 样本量 | 说明 |
|--------|--------|------|
| `engineered_data.csv` | 421 位名人 | 原始特征数据 |
| `mcmc_smooth_results.csv` | 368 周 | MCMC估计的粉丝投票（含决赛周） |

### 2.2 特征工程

#### 目标变量
- **评委评分** (`avg_judge_score`)：名人在赛季中的平均周得分
- **粉丝投票** (`avg_fan_vote`)：MCMC估计的平均粉丝投票份额

#### 特征变量
| 特征 | 类型 | 说明 |
|------|------|------|
| `celebrity_age_during_season` | 数值 | 参赛时年龄 |
| `season` | 数值 | 赛季编号 (1-34) |
| `partner_hist_score_mean_loo` | 数值 | Partner历史平均得分（留一法） |
| `partner_hist_placement_loo` | 数值 | Partner历史平均名次（留一法） |
| `partner_n_celebs_loo` | 数值 | Partner带过的其他名人数量 |
| `celebrity_industry` | 分类 | 名人所属行业（独热编码，26个类别） |
| `is_usa` | 二值 | 是否为美国本土选手 |
| `is_california` | 二值 | 是否来自加利福尼亚州 |
| `us_region` | 分类 | 美国选手所属区域（West/South/Midwest/Northeast/Other） |

### 2.3 数据泄露问题与修正

**问题发现**：初始版本直接使用Partner的整体统计数据（包含当前名人），导致数据泄露。

例如：某Partner只出场1次，其`partner_hist_score_mean`就等于该名人自己的得分，相当于用目标变量预测目标变量。

**修正方案**：采用**留一法 (Leave-One-Out)**
- 计算每位名人的Partner统计时，排除该名人自己
- 对于只出场1次的Partner（无法计算留一统计），使用全局均值填充
- 共有20位名人使用了全局均值填充

## 3. 模型选择实验

### 3.1 候选模型

我们评估了两种回归模型：

| 模型 | 优势 | 劣势 |
|------|------|------|
| **Ridge回归** | 系数可解释、方向明确、抗共线性 | 只能捕捉线性关系 |
| **随机森林** | 可捕捉非线性、特征交互 | 易过拟合、不可解释方向 |

### 3.2 评估指标

采用**5折交叉验证 R²** 作为主要评估指标，避免过拟合导致的虚高性能。

### 3.3 实验结果

#### 评委评分预测

| 模型 | R² (训练集) | R² (5折CV) | 判断 |
|------|-------------|------------|------|
| Ridge回归 | 0.4165 | **0.2975 ± 0.10** | ✅ 泛化较好 |
| 随机森林 | 0.8902 | 0.2074 ± 0.08 | ⚠️ 过拟合 |

#### 粉丝投票预测

| 模型 | R² (训练集) | R² (5折CV) | 判断 |
|------|-------------|------------|------|
| Ridge回归 | 0.1467 | **-0.0313 ± 0.01** | 无法预测 |
| 随机森林 | 0.7510 | -0.0114 ± 0.05 | ❌ 严重过拟合 |

### 3.4 模型选择结论

**选择 Ridge 线性回归**，理由如下：

1. **泛化能力**
   - 随机森林训练R²=0.89，但CV仅0.21，过拟合严重
   - Ridge回归训练/CV差距更小，泛化更稳定

2. **可解释性**
   - Ridge系数有正负方向，可直接解读因素影响方向
   - 随机森林只有重要性，无法判断正向还是负向影响

3. **适用性**
   - 粉丝投票本质不可预测（R²≈0），复杂模型无意义
   - 简单模型足以揭示"不可预测"这一核心结论

## 4. 最终模型结果

### 4.1 模型配置

```python
from sklearn.linear_model import Ridge
from sklearn.preprocessing import StandardScaler

model = Ridge(alpha=1.0)  # L2正则化
scaler = StandardScaler()  # 特征标准化
```

### 4.2 核心发现

#### 模型性能对比

| 目标变量 | R² (5折CV) | 解释 |
|----------|------------|------|
| 评委评分 | **0.2975** | 可被观测因素部分解释 |
| 粉丝投票 | **-0.0313** | 完全无法预测 |

#### 标准化回归系数对比

| 特征 | 评委评分系数 | 粉丝投票系数 | 解读 |
|------|-------------|-------------|------|
| **年龄** | **-0.63** | -0.01 | 评委偏好年轻选手，粉丝不偏好 |
| **赛季** | +0.03 | -0.01 | 赛季效应较弱 |
| **Partner得分** | **+0.27** | -0.01 | 评委认可优秀Partner，粉丝不在意 |
| **Partner名次** | +0.10 | -0.01 | 影响较弱 |
| **Partner经验** | +0.16 | -0.002 | 评委偏好资深Partner |
| **是否美国人** | -0.02 | +0.002 | 国籍影响极弱 |
| **是否加州** | **-0.16** | -0.01 | 加州选手评委评分较低 |

#### 地区效应分析 (新增)

**数据分布**：
- 美国本土选手：365位 (86.7%)
- 国际选手：56位 (13.3%)

**美国选手区域分布**：
| 区域 | 选手数 | 占比 |
|------|--------|------|
| South (南部) | 115 | 31.5% |
| West (西部) | 99 | 27.1% |
| Northeast (东北部) | 84 | 23.0% |
| Midwest (中西部) | 62 | 17.0% |
| Other | 5 | 1.4% |

**加州效应**：
- 加州选手：72位 (17.1%)
- 加州选手评委评分系数：**-0.16**（显著负向）
- 解读：加州选手（多为好莱坞娱乐明星）评委评分较低，可能因为他们的舞蹈基础相对较弱

**区域回归系数**：
| 区域 | 评委评分系数 | 粉丝投票系数 |
|------|-------------|-------------|
| West | +0.03 | +0.001 |
| South | +0.01 | +0.001 |
| Midwest | -0.01 | -0.005 |
| Northeast | -0.05 | +0.003 |
| Unknown (国际) | +0.02 | -0.002 |

**地区效应结论**：
1. **对评委评分**：区域影响较弱（系数均在±0.05以内），东北部略有负面影响
2. **对粉丝投票**：区域影响极弱（系数均在±0.005以内），无法解释粉丝投票差异
3. **加州效应显著**：加州选手评委评分明显较低（-0.16），但粉丝投票无差异
4. **假设验证**：南部/中西部选手并未表现出更强的粉丝支持（系数接近0）

**关键发现**：

### 4.3 结论

1. **评委评分可预测 (R²=29.8%)**
   - 主要驱动因素：年龄 > Partner历史表现 > 加州效应
   - 评委关注"舞蹈能力相关因素"
   - 演艺背景有正面效应，喜剧/政治背景有负面效应
   - 加州选手（好莱坞明星）评委评分较低

2. **粉丝投票不可预测 (R²≈-3%)**
   - 所有观测因素都无法解释粉丝投票
   - 包括地区因素也无法解释
   - 粉丝投票由"难以量化的个人魅力"驱动

3. **核心差异**
   - Partner效应：评委高度认可（+0.27），粉丝完全忽视（≈0）
   - 年龄效应：评委惩罚高龄（-0.63），粉丝无差异（≈0）
   - 加州效应：评委评分较低（-0.16），粉丝无差异（≈0）
   - 地区效应：对两者影响均极弱（所有区域系数<0.05）
   - 行业效应：评委有系统性偏好，粉丝无差异

## 5. 方法论总结

### 5.1 技术路线

```
原始数据 → 留一法特征工程 → 模型选择实验 → Ridge回归分析 → 对比结论
```

### 5.2 关键方法论决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| Partner统计 | 留一法 | 避免数据泄露 |
| 行业编码 | 独热编码 | 无序分类变量 |
| 地区编码 | 区域划分 + 独热编码 | 将50个州简化为5个区域 |
| 特征缩放 | StandardScaler | 使系数可比较 |
| 正则化 | Ridge (L2) | 抗共线性 |
| 模型选择 | Ridge vs 随机森林 | 基于CV性能 |
| 评估指标 | 5折CV R² | 避免过拟合 |

### 5.3 局限性

1. **粉丝投票为估计值**：来自MCMC模型，非真实观测
2. **行业分类不均衡**：部分行业样本量<5
3. **地区分类粗粒度**：将50个州简化为5个区域，可能丢失细粒度信息
4. **国际选手**：无州级地区信息，统一归类为Unknown
5. **时序未建模**：未考虑赛季内的动态变化
